# TechCon 2024 Conference Website

## Project Overview

The **TechCon 2024 Conference Website** is a multi-page HTML project designed to demonstrate practical knowledge of semantic HTML, accessibility, multimedia integration, navigation structures, and user form handling. This project consists of **five core webpages**, each representing a crucial part of a real-world conference website.

## Project Structure

**GitHub Repository:** `TechCon_HTML`
**Project Directory:** `techcon_website`
**Files Included:**

* `index.html` (Homepage)
* `about.html`
* `schedule.html`
* `register.html`
* `contact.html`

## Pages Description

### 1. Homepage (`index.html`)

* Introduces TechCon 2024.
* Contains a header with conference title and navigation menu.
* Includes a welcome section with a promotional video.
* Features a footer with copyright information.

### 2. About Page (`about.html`)

* Provides the history, mission, and goals of TechCon.
* Highlights notable speakers.

### 3. Schedule Page (`schedule.html`)

* Displays the full event schedule using a semantic HTML table.
* Includes sessions, speakers, times, and event descriptions.

### 4. Register Page (`register.html`)

* Contains a user-friendly registration form.
* Includes text inputs, email input, password field, checkboxes, and a submit button.

### 5. Contact Page (`contact.html`)

* Displays contact information and social media links.
* Includes a "Contact Us" form.
* Embeds a Google Map for location reference.

## Concepts Covered

### **1. Semantic HTML Structure**

* Use of `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>`.

### **2. Navigation & Linking**

* Internal linking using `<a>` tags for smooth page navigation.

### **3. Multimedia Integration**

* Embedding videos using `<video>`.
* Embedding Google Maps using `<iframe>`.

### **4. Tables**

* Structured schedules built with `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, and `<td>`.

### **5. Forms**

* Implementing registration and contact forms using `<form>` elements and various input types.

### **6. Accessibility Considerations**

* Proper use of `alt` text on images.
* Semantic tag usage for better screen reader support.
* Video description fallback.

## How to Run the Project

1. Clone or download the repository.
2. Navigate to the `techcon_website` directory.
3. Open any HTML file in a browser (Chrome, Firefox, Edge, etc.).

Example:

```
index.html
```
